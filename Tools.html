DefSound: Установки аудио устройства по умолчанию



Утилита для переключения аудио устройства по-умолчанию. Программа имеет простой GUI-интерфейс, который выражается в висящей в трее иконке. В контекстном меню этой иконки можно переключить роли аудиоустройств в системе. Так же утилита воспринимает параметры командной строки (например, для написания скриптов).

Системные требования: ОС Windows Vista и старше
