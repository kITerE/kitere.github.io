Ниже приведены некоторые параметры командной строки отладчика:



WinDbg.exe -I

Установка WinDbg отладчиком по-умолчанию



WinDbg -z DUMP_FILE

Открытие dump-файла падения с именем DUMP_FILE. В качестве файла можно указать обыкновенный PE-файл



Иногда бывает нужно скопировать файл с символами в директорию с исполняемым файлом. Конечно, можно загрузить файл как dump-файл, набрать команду lm и скопировать полное имя файла с символами. Но можно немного автоматизировать этот процесс. Поэтому можно создать команду для FAR’а, которая выглядит так:


@start WinDbg -z !.! -c ".reload; .copysym .\; q"

Где !.! - метасимвол FAR’а, интерпретируемый как полный путь к выбранному файлу. А для копирования файла символов использована команда WinDbg .copysym. Важно помнить, что WinDbg не перезаписывает существующий файл символов при копировании.
